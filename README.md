
# Docomment

Docomment 是一个类似doxygen的python文档整理工具，它可以帮你把指定目录下的所有.py文件中的说明提取出来，整理成markdown文档
保存在你指定的目录下。

## 适用脚本：python3

## 文档中的解释的书写格式：samply.py

## 用法
python3 crtmd.py sourceDir destincationDir


## 更新进度：version 0.0.1 完成基本功能
